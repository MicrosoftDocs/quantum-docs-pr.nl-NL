# <a name="q-libraries"></a>Q#-bibliotheken

De Q#-standaardbibliotheek omvat bewerkingen, functies en door gebruiker gedefinieerd typen die worden gebruikt om kwantumprogramma's te ontwikkelen. Drie extra bibliotheken die zijn gebouwd op de standaardbibliotheek zijn ook beschikbaar: de chemische, numerieke en machine learning-bibliotheken.

Raadpleeg <xref:microsoft.quantum.libraries> voor meer informatie over de Q#-bibliotheken.
