# <a name="tail-function"></a>Functie staart

Naam ruimte: [micro soft. Quantum. arrays](xref:Microsoft.Quantum.Arrays)

Pakket [](https://nuget.org/packages/)


Retourneert het laatste element van de matrix.

```qsharp
function Tail<'A> (array : 'A[]) : 'A
```


## <a name="input"></a>Invoer

### <a name="array--a"></a>matrix: ' A []

De matrix waarvan het laatste element is gemaakt. De matrix moet een lengte van ten minste 1 hebben.



## <a name="output--a"></a>Uitvoer: ' A

Het laatste element van de matrix.

## <a name="type-parameters"></a>Type parameters

### <a name="a"></a>' A

Het type van de matrix elementen.
