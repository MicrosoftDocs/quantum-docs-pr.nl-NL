# <a name="timescp-function"></a>De functie TimesCP

Naam ruimte: [micro soft. Quantum. math](xref:Microsoft.Quantum.Math)

Pakket [](https://nuget.org/packages/)


Retourneert het product van twee invoer waarden.

```qsharp
function TimesCP (a : Microsoft.Quantum.Math.ComplexPolar, b : Microsoft.Quantum.Math.ComplexPolar) : Microsoft.Quantum.Math.ComplexPolar
```


## <a name="input"></a>Invoer

### <a name="a--complexpolar"></a>a: [ComplexPolar](xref:Microsoft.Quantum.Math.ComplexPolar)

De eerste invoer $a $ die moet worden vermenigvuldigd.


### <a name="b--complexpolar"></a>b: [ComplexPolar](xref:Microsoft.Quantum.Math.ComplexPolar)

De tweede invoer $b $ om te worden vermenigvuldigd.



## <a name="output--complexpolar"></a>Uitvoer: [ComplexPolar](xref:Microsoft.Quantum.Math.ComplexPolar)

Het product $a \times b $.
