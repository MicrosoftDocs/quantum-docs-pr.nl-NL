# <a name="snd-function"></a>De functie snd

Naam ruimte: [micro soft. Quantum. Canon](xref:Microsoft.Quantum.Canon)

Pakket: [micro soft. Quantum. Standard](https://nuget.org/packages/Microsoft.Quantum.Standard)


Retourneert het tweede element aan de hand van een paar.

```qsharp
function Snd<'T, 'U> (pair : ('T, 'U)) : 'U
```


## <a name="input"></a>Invoer

### <a name="pair--tu"></a>paar: ('T, ' U ')

Een tuple met twee elementen.



## <a name="output--u"></a>Uitvoer: ' U

Het tweede element van `pair` .

## <a name="type-parameters"></a>Type parameters

### <a name="t"></a>T

Het type van het eerste lid van het paar.
### <a name="u"></a>' U

Het type van het tweede lid van het paar.
