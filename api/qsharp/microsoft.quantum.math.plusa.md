# <a name="plusa-function"></a>Functie plus

Naam ruimte: [micro soft. Quantum. math](xref:Microsoft.Quantum.Math)

Pakket [](https://nuget.org/packages/)


Retourneert de som (samen voeging) van twee invoer waarden.

```qsharp
function PlusA<'Element> (a : 'Element[], b : 'Element[]) : 'Element[]
```


## <a name="input"></a>Invoer

### <a name="a--element"></a>a:-element []

De eerste invoer $a $ die moet worden opgeteld.


### <a name="b--element"></a>b: element []

De tweede invoer $b $ moet worden opgeteld.



## <a name="output--element"></a>Output: ' element []

De som $a + b $.

## <a name="type-parameters"></a>Type parameters

### <a name="element"></a>Element

Het type van elk element in elk van de twee invoer matrices.
