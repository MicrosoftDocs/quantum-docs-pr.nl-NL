# <a name="enabletestingvianame-user-defined-type"></a>Door de gebruiker gedefinieerd EnableTestingViaName-type

Naam ruimte: [micro soft. Quantum. Diagnostics](xref:Microsoft.Quantum.Diagnostics)

Pakket: [micro soft. Quantum. QSharp. core](https://nuget.org/packages/Microsoft.Quantum.QSharp.Core)


Een door het compiler herkend kenmerk waarmee een alternatieve naam kan worden gedefinieerd die kan worden gebruikt bij het laden van een type of aanroepable voor test doeleinden.

```qsharp

@ Microsoft.Quantum.Core.Attribute()
newtype EnableTestingViaName = (String);
```
