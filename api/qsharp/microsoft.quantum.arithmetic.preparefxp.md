# <a name="preparefxp-operation"></a>Bewerking PrepareFxP

Naam ruimte: [micro soft. Quantum. aritmetische](xref:Microsoft.Quantum.Arithmetic)

Pakket: [micro soft. Quantum. numerics](https://nuget.org/packages/Microsoft.Quantum.Numerics)


Initialiseer een Quantum getal met een vaste komma naar een klassieke constante.

```qsharp
operation PrepareFxP (constant : Double, fp : Microsoft.Quantum.Arithmetic.FixedPoint) : Unit is Adj + Ctl
```


## <a name="input"></a>Invoer

### <a name="constant--double"></a>constante: [dubbel](xref:microsoft.quantum.lang-ref.double)

Een constante waarmee het Quantum getal met een vast punt moet worden geïnitialiseerd.


### <a name="fp--fixedpoint"></a>FP: [FixedPoint](xref:Microsoft.Quantum.Arithmetic.FixedPoint)

Vast punt nummer (van het type FixedPoint) dat moet worden geïnitialiseerd.
