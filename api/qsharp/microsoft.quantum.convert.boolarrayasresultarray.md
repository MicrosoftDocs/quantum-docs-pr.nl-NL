# <a name="boolarrayasresultarray-function"></a>De functie BoolArrayAsResultArray

Naam ruimte: [micro soft. Quantum. Convert](xref:Microsoft.Quantum.Convert)

Pakket: [micro soft. Quantum. Standard](https://nuget.org/packages/Microsoft.Quantum.Standard)


Converteert een `Bool[]` type naar een `Result[]` type, waarbij `true` wordt toegewezen aan `One` en `false` wordt toegewezen aan `Zero` .

```qsharp
function BoolArrayAsResultArray (input : Bool[]) : Result[]
```


## <a name="input"></a>Invoer

### <a name="input--bool"></a>invoer: [BOOL](xref:microsoft.quantum.lang-ref.bool)[]

`Bool[]` worden geconverteerd.

Een `Result[]` vertegenwoordigt de `input` .
