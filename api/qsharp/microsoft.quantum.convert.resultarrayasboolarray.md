# <a name="resultarrayasboolarray-function"></a>De functie ResultArrayAsBoolArray

Naam ruimte: [micro soft. Quantum. Convert](xref:Microsoft.Quantum.Convert)

Pakket: [micro soft. Quantum. Standard](https://nuget.org/packages/Microsoft.Quantum.Standard)


Converteert een `Result[]` type naar een `Bool[]` type, waarbij `One` wordt toegewezen aan `true` en `Zero` wordt toegewezen aan `false` .

```qsharp
function ResultArrayAsBoolArray (input : Result[]) : Bool[]
```


## <a name="input"></a>Invoer

### <a name="input--__invalidresult__"></a>invoer: __ongeldig <Result>__[]

`Result[]` worden geconverteerd.



## <a name="output--bool"></a>Uitvoer: [BOOL](xref:microsoft.quantum.lang-ref.bool)[]

Een `Bool[]` vertegenwoordigt de `input` .
