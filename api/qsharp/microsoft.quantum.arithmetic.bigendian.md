Naam ruimte: [micro soft. Quantum. aritmetische](xref:Microsoft.Quantum.Arithmetic)

Pakket [](https://nuget.org/packages/)


Registreert dat een niet-ondertekend geheel getal codeert in de volg orde big endian. De Qubit met index `0` codeert de hoogste bit van een niet-ondertekend geheel getal.

```qsharp

newtype BigEndian = (Qubit[]);
```
