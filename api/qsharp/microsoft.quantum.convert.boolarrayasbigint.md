Naam ruimte: [micro soft. Quantum. Convert](xref:Microsoft.Quantum.Convert)

Pakket [](https://nuget.org/packages/)


Hiermee wordt een opgegeven matrix met Boole-waarden geconverteerd naar een equivalent Big-geheel getal.
Het element 0 van de matrix is de minst significante bit van het grote geheel getal.
