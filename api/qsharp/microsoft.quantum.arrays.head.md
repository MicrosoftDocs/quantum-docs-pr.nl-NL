# <a name="head-function"></a>Head-functie

Naam ruimte: [micro soft. Quantum. arrays](xref:Microsoft.Quantum.Arrays)

Pakket [](https://nuget.org/packages/)


Retourneert het eerste element van de matrix.

```qsharp
function Head<'A> (array : 'A[]) : 'A
```


## <a name="input"></a>Invoer

### <a name="array--a"></a>matrix: ' A []

De matrix waarvan het eerste element wordt gemaakt. De matrix moet een lengte van ten minste 1 hebben.



## <a name="output--a"></a>Uitvoer: ' A

Het eerste element van de matrix.

## <a name="type-parameters"></a>Type parameters

### <a name="a"></a>' A

Het type van de matrix elementen.
