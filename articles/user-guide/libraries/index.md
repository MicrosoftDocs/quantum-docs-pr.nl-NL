# <a name="overview-of-no-locq-libraries"></a>Overzicht van Q#-bibliotheken
De Quantum Development Kit is voorzien van verschillende bibliotheken die de ontwikkeling van kwantumtoepassingen in Q# vergemakkelijken.
In deze sectie van de documentatie beschrijven we deze bibliotheken en hoe ze in uw programma's kunnen worden gebruikt.

- [**Standaardbibliotheken**](xref:microsoft.quantum.libraries.standard.intro): Deze sectie beschrijft de prelude, die de interface tussen Q#-programma's en doelmachines definieert, evenals de canon, een Q#-bibliotheek met algemene bewerkingen en functies voor het schrijven van Q#-programma's.
- [**Bibliotheek voor kwantumchemie**](xref:microsoft.quantum.chemistry.concepts.intro): Deze sectie beschrijft de bibliotheek voor kwantumchemie, die een datamodel bevat voor het laden van representaties van fermionische hamiltonianen, evenals bewerkingen en functies voor kwantumsimulaties die op deze representaties toepasbaar zijn.
- [**Bibliotheek voor kwantumberekeningen**](xref:microsoft.quantum.numerics.intro): Deze sectie beschrijft de bibliotheek voor kwantumberekeningen, die implementaties bevat voor een groot aantal wiskundige functies. Deze bibliotheek ondersteunt representaties met gehele getallen (met en zonder tekens) en vaste punten.
- [**Kwantumbibliotheek voor machine learning**](xref:microsoft.quantum.machine-learning.concepts.intro): In deze sectie wordt de kwantumbibliotheek voor machine learning beschreven. Bronnen van de bibliotheken en codevoorbeelden zijn te vinden op GitHub.
Zie [Licenties](xref:microsoft.quantum.libraries.licensing) voor meer informatie. Voor de bibliotheken zijn ook pakketverwijzingen ('binaire bestanden') beschikbaar. Deze bieden een alternatieve manier om de bibliotheken in projecten op te nemen.
Een handige manier om deze pakketverwijzingen te verkrijgen, is via [NuGet](https://nuget.org).
